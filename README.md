# Recipe Recommendation Data Pipeline

## Overview
This project demonstrates an end-to-end data pipeline that ingests recipe data and grocery inventory data, stores them in a structured database, and generates meal recommendations based on available ingredients.

The goal is to showcase practical data engineering and analytics skills, including data ingestion, cleaning, modeling, and querying.

## Project Goals
- Ingest recipe data from public sources
- Ingest grocery inventory data (simulated or user-provided)
- Normalize and store data in a relational database
- Match recipes to available ingredients
- Enable simple querying and recommendations

## Planned Tech Stack
- Python
- SQL (PostgreSQL or SQLite)
- Pandas
- SQLAlchemy
- GitHub
- Optional: Airflow or scheduled jobs
